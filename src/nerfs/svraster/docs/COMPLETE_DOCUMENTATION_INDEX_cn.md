# SVRaster 完整技术文档索引（重构更新版）

## 🎯 文档概述

SVRaster 1.0.0（Sparse Voxel Rasterization）是一种先进的稀疏体素渲染技术，采用**双渲染器架构**设计。本文档集提供了简洁、实用的技术指南，专注于最新的重构架构。

## 🔄 重构核心变化

### 双渲染器架构
- **训练阶段**: `VolumeRenderer` - 体积渲染用于梯度传播
- **推理阶段**: `VoxelRasterizer` - 光栅化用于快速渲染

### 紧密耦合设计
- `SVRasterTrainer` ↔ `VolumeRenderer`
- `SVRasterRenderer` ↔ `VoxelRasterizer`
